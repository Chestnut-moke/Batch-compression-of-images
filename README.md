# 图片批量压缩工具 (Image Batch Compression Tool)

这是一个基于 Python 和 Tkinter 开发的轻量级图片批量压缩工具，旨在提供简单、高效的图片处理体验。它支持多种压缩策略，包括分辨率调整和精确的文件大小控制。

## ✨ 主要功能

- **批量处理**：一次选择多张图片进行并行化处理思路。
- **灵活导出**：可自定义导出路径和文件名后缀（如 `_min`），防止覆盖原图。
- **多格式支持**：支持导出为 `JPEG`、`PNG`、`WEBP` 或保持 `原格式`。
- **智能缩放**：支持设置最大宽度或高度，采用高性能的 Lanczos 采样算法进行等比例缩放。
- **精确大小控制**：针对 JPEG 和 WEBP，支持设置目标文件大小 (KB)，程序将通过二分法自动寻找最佳压缩质量。
- **图形化界面**：直观的 GUI 界面，操作简单，带进度条显示。

## 🚀 快速开始

### 方式一：直接运行 (推荐)
如果你已经安装了 Python 环境，可以直接运行源代码：

1. 安装依赖：
   ```bash
   pip install Pillow
   ```
2. 运行程序：
   ```bash
   python image_compressor.py
   ```

### 方式二：使用可执行文件
在 `dist/` 目录下找到 `image_compressor.exe`，直接双击即可运行（无需安装 Python）。

## 🛠️ 技术栈
- **语言**: Python 3
- **GUI 框架**: Tkinter (内置)
- **图像处理**: Pillow (PIL)

## 📝 使用说明
1. 点击 **选择图片** 按钮选取一张或多张图片。
2. 设置 **导出路径**（默认为原图目录）。
3. (可选) 修改 **自动添加后缀**。
4. (可选) 在 **限制参数** 中设置你想要的分辨率或文件大小限制。
5. 点击 **开始批量压缩**，等待进度条完成即可。
